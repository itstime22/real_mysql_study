### 진행방식
- 매 주마다 범위를 설정하여 책을 읽습니다. 각자 읽은 내용을 자신만의 방식으로 정리합니다.

- 내용을 정리한 파일과 질문들을 깃허브에 푸시합니다.

### 스터디 진행
- 발표자는 각자 준비한 정리 파일을 바탕으로 읽은 내용을 정리하여 발표합니다.
- 깃허브에 올린 질문들을 중심으로, 함께 답변을 찾아가며 토의합니다.
- 대표자는 질문들에 대한 답변들을 정리하여, `week{n}/summary.md` 에 정리합니다.
